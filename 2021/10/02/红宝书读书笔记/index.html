第一章 什么是 JavaScript

JavaScript 是一门用来与网页交互的脚本语言，包含三个部分：

  1. ECMAScript；
  2. 文档对象模型（DOM）；
  3. 浏览器对象模型（BOM）。

第二章 HTML 中的 JavaScript

  1. 所有<script>元素会依照它们在网页中出现的次序被解释。在不使用defer和async属性的情况下，包含在<script>元素中的代码必须严格按次序解释；
  2. 对不推迟执行的脚本，浏览器必须解释完位于<script>元素中的代码，然后才能继续渲染页面剩余部分。为此，通常应该把<script>元素放在页面末尾，介于主内容之后及</body>标签之前。

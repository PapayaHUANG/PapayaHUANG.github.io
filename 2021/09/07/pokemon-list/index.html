pokemon-list

2021-09-07

项目目的

这个项目是跟着John Smilga的React Course练习的，这个项目主要是练习useState。

项目效果

  • 模块化在宝可梦清单添加宝可梦的信息
  • 宝可梦信息包括姓名、照片、属性
  • 宝可梦清单标题显示清单宝可梦数量
  • 宝可梦清单提供一键删除所有宝可梦的功能

实现过程

按信息从下往上来梳理：

  1. Data.js 一个数据的数组，包含所有宝可梦的信息
  2. Card.js 模块化提取数组中宝可梦的信息
  3. App.js 遍历所有宝可梦信息，并按照固定格式呈现出来

因为React的模块化是通过dripple来实现的，所以在代码编写的时候，除了信息源Data.js之外，其他代码的编写顺序和上述清单相反。

  • Data.js
    在数组清单的编写中需要注意的是除了每一只宝可梦的基本信息，还需要给每一只宝可梦加上id：

    {
  id: 1,//添加id
  img: 'https://upload.wikimedia.org/wikipedia/en/a/a6/Pok%C3%A9mon_Pikachu_art.png',
  name: 'Pikachu',
  type: 'Electric',
},

    这个id是将数组信息向上传递的关键，是key，相当于每一个宝可梦的身份证，详情可以看官方文档

  • App.js
    在这个部分我们需要获取Data.js里所有宝可梦信息，并且把这些信息分类处理：

    import React from 'react';
import { Data } from './data';
const App = () => {
let [value, setValue] = React.useState(Data);

return (
  <section className="container">
    <h3 className="title">You have {value.length} Pokemon(s) at hand</h3>
    <article>
      {value.map((card) => {
        return <Card key={card.id} card={card}></Card>;
      })}
    </article>
  );
};

    因为这个页面还要实现点击按钮，清除所有宝可梦信息，涉及到状态的改变，所有这里我们使用了React hook中的useState()；
    页面加载的初始状态是呈现出Data.js中数组的所有信息，所以我们可以设置初始状态value为引入的Data，并使用map()的方法遍历。在这一步我们需要提取出每一个card的id，和单个对象，并把这些信息传递给Card方便之后进一步分解使用。

    <button type="button" className="btn" onClick={() => setValue([])}>
      All Back to Pokeball
    </button>

    点击按钮后清单晴空，所以我们把setValue的值设定为空数组[];

  • Card.js
    在这个部分我们要接收App.js传导过来的property，并分解：

    const Card = (props) => {
const [value, setValue] = React.useState(props.card);
const { img, name, type } = value;

return (
  <article className="cards">
    <img src={img} alt="" className="img"></img>
    <div className="info">
      <h3 className="name">Name: {name}</h3>
      <h5 className="type">Type: {type}</h5>
    </div>
  </article>
);
};

    这样加上样式就完成了一个简单的宝可梦清单。

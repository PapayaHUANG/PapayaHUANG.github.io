CSS样式的继承

2020-04-21

CSS的全称为Cascading Style Sheet，所以Cascading（叠层）是理解CSS一个很基础和关键的概念。

所谓的“叠层”，就是说CSS设定的样式是可以叠加的，具体一点即：样式可以被继承，或者被覆盖。而叠层的顺序遵循一个重要的逻辑就是“子承父性”。

谁是子谁是父涉及到HTML标签的嵌套，可以打破HTML嵌套框架的方式是添加class或者id属性。

在使用CSS设定元素属性的时候，我们把以HTML标签为基础的元素命名为“元素选择器”；把class元素命名为“类选择器”；而id元素命名为“ID选择器”。

在CSS处理叠层的优先级时，除了要考量以上三个选择器，还要把<style>属性内的声明纳入在内，则<style> > ID > 类 > 元素。

MDN是用4位数来计算各种选择器的优先级的。MDN原文

除此之外需要额外注意的点是，CSS的继承不是子承父的所有属性，一般情况来说，如果样式会影响文本的外观会被继承，其他属性不能被继承，如border。

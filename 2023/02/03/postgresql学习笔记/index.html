数据库结构与数据查看

从大到小分别是：

  • database
  • table
  • column
  • detail

对应的查看这些单元的命令行是：

\l

该命令可以展开看有哪些数据库

\c database_name
\d

链接到某个数据库之后，就可以查看数据库中有哪些列表（table）；
查看table内有哪些column也可以使用\d；
查看一个 table的详细数据使用：

SELECT * FROM table_name;

创建 database 或者 table

使用CREATE DATABASE命令，注意如果是创建 table 需要写成CREATE DATABASE table_name()。之所以多了一个空格是因为在空格内部可以直接填写TABLE包含的COLUMN，一起创建。

修改 table 的内容

使用ALTER TABLE命令来实现：

  • 添加列使用ADD COLUMN;
  • 删除列使用DROP COLUMN;
  • 修改列的名字使用RENAME COLUMN TO ;;

使用INSERT INTO命令来实现：
行内容的添加，具体命令是INSERT INTO table_name() VALUES();，注意可以同时在一条命令内添加多个行。

primary key 和 foreign key

每一个表格都应该有一个 primary key，而 foreign key 是链接两个不同的表格之间的关键行。

表格与表格之间的关系

有两种联系方式，一对一或者多对多。
一对一可以通过 foreign key 来实现，而多对多可以单独创建一个表格（junction table），然后在这个表格中添加两个 primary key，只做链接作用。
若想在命令行内查看不同表格之间的联系，使用到的命令是JOIN:

  • 一对一
    SELECT columns FROM table-1 FULL JOIN table-2 ON table-1.something=table-2.something;;
  • 多对多
    SELECT columns FROM junction-table FULL JOIN table-1 ON junction-table.something=table-1.something FULL JOIN table-2 ON junction-table.something=table-2.something;

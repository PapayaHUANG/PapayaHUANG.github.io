整理HTML中的块元素和内联元素

2021-04-20

前言

最近在读《Head First》，这本书很好地补充了我对 HTML 和 CSS 框架知识的缺乏。HTML 提供一个网页的结构，而 CSS 提供网页的表现。了解不同的 HTML 元素属性对后期 CSS 做页面布局的影响很大，在此总结一下 HTML 中常见元素的属性。

块元素（block element）

在呈现上是独立的一块，单独为一行

内联元素（in-line element）

在呈现上前后两个元素不会换行，直至一行完全占满

常见的块元素

  • <h1>~<h6>

  • <div>

  • <p>

  • <li><ul><ol>

  • <blockquote>

  • <table>

常见的内联元素有

  • <em>

  • <strong>

  • <img>

  • <q>

  • <span>

  • <label>

  • <input>

特殊情况

  • <a>

看起来像内联元素，但可以包含块元素，所以需要根据具体情景来判断；

  • <br>

处在块元素和内联元素的模糊地段，可以产生换行，但不会像块元素<p>那样，把两端文本分成独立的两块。

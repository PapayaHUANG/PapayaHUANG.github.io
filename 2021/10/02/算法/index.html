题目

Factorialize a NumberPassed
Return the factorial of the provided integer.

If the integer is represented with the letter n, a factorial is the product of all positive integers less than or equal to n.

Factorials are often represented with the shorthand notation n!

For example: 5! = 1 _ 2 _ 3 _ 4 _ 5 = 120

Only integers greater than or equal to zero will be supplied to the function.

解法

for 循环

function factorialize(num) {
  for (var product = 1; num > 0; num--) {
    product *= num;
  }
  return product;
}

factorialize(5); //120

注解：
这里需要使用var在函数内声明变量，因为要利用var的变量提升。

递归

function factorialize(num) {
  if (num === 0) {
    return 1;
  } else {
    return num * factorialize(num - 1);
  }
}

注解：
传统的递归是先执行递归调用，然后获取递归调用的返回值并进行计算结果，所以在每次递归调用返回之前，不会得到计算结果。

reduce()

function factorialize(num) {
  return num < 0
    ? 1
    : new Array(num)
        .fill(undefined)
        .reduce((product, val, index) => product * (index + 1), 1);
}

注解：
在这里创建用new Array()了一个新的数组,并且设定这个数组的长度等于num,因为要对这个数组采用reduce()的方法，所以先用undefined填充了数组。

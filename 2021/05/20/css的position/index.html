position

2021-05-20

最近在做练习的时候常用到position来定位元素，所以在此结合我看的资料来整理一下postion的各个属性。

position的属性有5个值：

  • static
  • relative
  • fixed
  • absolute
  • sticky

可以给这5个值做一个分类，static独自为一类，relative/fixed/absolute为一类，sticky为一类。

第一类：默认值

  • static

static为position的默认值。

当值为static或者未设定position值时，元素的位置遵循页面流（normal flow）。

此时top/bottom/left/right的值是无效的。

第二类：相对值

  • relative

relative相对的是默认位置；

必须搭配top/bottom/left/right一起使用

  • absolute
    absolute相对于上级元素（一般是父元素）进行偏移；
    定位的基点不能是static，如果向上检索的父元素都为static，则定位基点为网页根元素html；
    所以通常要使用absolute的话，会把父元素的position设定为relative；
    absolute定位的元素会忽略页面流，周边元素不受影响；
    可以搭配top/bottom/left/right一起使用;

  • fixed
    fixed相对的是视窗（viewport），这样会导致元素的位置不随着页面的滚动而变化，好像固定在网页上一样；

可以搭配top/bottom/left/right一起使用;
若不设定这四个属性，则固定在元素的初始位置。

  • 第三类 产生动态效果值
    sticky
    sticky是2017才加入的新值；
    sticky的效果很像relative和fixed的结合；

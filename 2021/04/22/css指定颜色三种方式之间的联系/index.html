CSS中三种指定颜色方式之间的联系

2021-04-22

前言

这里涉及到的 CSS 三种指定颜色的方式分别为：直接使用颜色英文名；设定 RGB 值；用十六进制码表示。

不论使用哪一种方式，都是告诉浏览器一个颜色中红、绿、蓝的分量分别是多少。

直接使用颜色英文名

CSS 有 16 个基本颜色，和 150 个拓展颜色。

16 个基本颜色如图

这些颜色名其实是预设了 RGB 的值

设定 RGB 值

设定 RGB 值有两个方法：
一个使用 0-100%的百分比来设定；
一个使用 0-255 的数值来设定。

举一个例子：
若想要 web 背景色为橙色，需要 80%的红色加上 40%的绿色。
其语法表达式可以写成：

body {
  background-color: rgb(80%, 40%, 0%);
}

也可以写成

body {
  background-color: rgb(204, 102, 0);
}

用 16 进制码表示颜色

用 0-255 来分别设定 rgb 的值是用十进制数值来表示颜色，我们可以把它改写成 16 进制数，上文的橙色就可以表示为：

body {
  background-color: #cc6600;
}

#CC6600从左往右看，前两位是 R 的值，中间两位是 G 的值，最后两位是 B 的值。
CC 换算成 10 进制就是：
1612+12=204
66 换算成 10 进制就是：
616+6=102
00 换算成 10 进制就是：
0*16+0=0

这就与rgb(204, 102, 0)完全一致。

当然#CC6600可以简写成#C60。

综上 CSS 表示颜色的方式其实都是来指定红、绿、蓝三原色的值。直接使用颜色名是直接使用已经设定好的名字，rgb 和 16 进制表达法可以通过计算相互转化。

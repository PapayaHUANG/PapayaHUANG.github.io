项目练习 —— 用flexbox练习写骰子面

2021-04-06

项目目的

用flex布局来排列骰子上面的黑点，写出骰子的6个面。

项目使用技能

flex相关语法中justify-content/align-items/align-self的运用。

基本思路

  1. 把骰子的面作为容器，点作为项目来排列；

  2. 因为flexbox是一维布局，仅基于主轴（main axis）和交叉轴（cross axis）两个坐标的变换来调整项目位置，而骰子上的黑点有1-6个不等，所以要先对不同点数的骰子面进行分类；

  3. 仅需一个空间进行排列组合：点数为1-3的骰子面可以分为一类；

  4. 需要两个空间进行排列组合：点数4-6的骰子面可以分为一类；

  5. 需要三个空间进行排列组合：点数5的骰子面单独为一类。

具体操作

一点骰子

   <div class="first-dice">
     <span class="dot"></span>
   </div>

   .first-dice{
      display: flex;
      align-items: center;
      justify-content: center;
    }
  • 该容器中只有一个项目(item)；
  • 通过justify-content和align-items在水平主轴和垂直交叉轴上让所有项目居中。

两点是骰子

   <div class="second-dice">
     <span class="dot"></span>
     <span class="dot"></span>
   </div>

   .second-dice \{
      display: flex;
      justify-content: space-between; 
    };
   .second-dice .dot:nth-child(2){
       align-self: flex-end;
    }
  • 该容器中有两个项目；
  • 先用space-between在水平方向让两个项目从中间分开，分别靠向容器的左侧和右侧；
  • 再用align-self调整第二个点，使其对齐容器底边。

三点骰子

   <div class="third-dice">
      <span class="dot"></span>
      <span class="dot"></span>
      <span class="dot"></span>
   </div>

   .third-dice {
      display: flex;
      justify-content: space-between;
    }
   .third-dice .dot:nth-child(2) {
      align-self:center;
    }
   .third-dice .dot:nth-child(3) {
      align-self: flex-end;
    }
  • 该容器中有三个项目；
  • 先用space-between确保水平方向三个点最左和左右分布在容器左右两侧；
  • 再用align-self分别将第二个点居中，第三个点分布在右下。

四点和六点骰子


   <div class="fourth-dice">
      <div class= "column">
         <span class="dot"></span>
         <span class="dot"></span>
   </div>
      <div class= "column">
         <span class="dot"></span>
         <span class="dot"></span>
      </div>
   </div>

   <div class="sixth-dice">
      <div class="column">
         <span class="dot"></span>
         <span class="dot"></span>
         <span class="dot"></span>
      </div>
      <div class="column">
         <span class="dot"></span>
         <span class="dot"></span>
         <span class="dot"></span>
      </div>
   </div>

   .fourth-dice, .sixth-dice {
      display: flex;
      justify-content: space-between
   }
   .fourth-dice .column, .sixth-dice .column {
      display: flex;
      flex-direction: column;
      justify-content: space-between;
   }
  • 因为4点和6点的骰子面可以看作以中间纵轴线对称，左右点数分布一致的图形，所以可以把4点和6点先分成2个一组的column和3个一组的column；
  • 先用space-between在水平主轴上让两个column分别靠向左侧和右侧；
  • 再在column内部，确保项目的排列方式是从上向下（column），再用space-between使两端项目靠近容器边界，项目之间平均分布。

五点骰子

   <div class="fifth-dice">
      <div class="column">
         <span class="dot"></span>
         <span class="dot"></span>
      </div>
      <div class="column">
         <span class="dot"></span>
      </div>
      <div class="column">
         <span class="dot"></span>
         <span class="dot"></span>
      </div>
   </div>

   .fifth-dice {
      display: flex;
      justify-content: space-between;
   }
   .fifth-dice .colum{
      display: flex;
      flex-direction: column;
      justify-content: space-between;
   }
   .fifth-dice .column:nth-child(2){
       justify-content: center;
   }
  • 5点的骰子面分布在四边的点的处理方式可以和4/6点的骰子一样，然后有一个点居中，这样就可以把5个点分成2-1-2三个column；
  • 先用space-between在水平轴上确保三个column分布开；
  • 再确保在column内部点是从上而下并且两端靠边平均分布；
  • 单拎出来第二个column，使点（项目）居中。（注意这里虽然只有一个项目，但是这里我们把这个项目理解为一个column里的唯一项目，所以在分布时我们用justify-content而非justify-self。

总结

  1. justify-content、align-items以及在这个项目里没有用到的align-content都是把一个容器内所有项目当作一个整体来进行排列分布，若需要调整其中一个项目，才使用align-self；
  2. 若项目的排列方向是上下方向，想象中的水平主轴相应也调成90度。

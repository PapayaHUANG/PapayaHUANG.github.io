前言

今天是 1024 程序员节，那么就用 HTML+CSS 画一个章鱼猫吧。

涉及技术

css 伪类

实现细节

HTML 结构

<div class="github-box">
  <div class="github">
    <i>
      <i>
        <i></i>
      </i>
    </i>
  </div>
</div>

其中github用于绘制黑色圆形背景，其他部分都是用::before和::after来拼凑出章鱼猫的脸、身子、耳朵、手以及爱心。

CSS 实现

所有的伪类都同意确定颜色、绝对位置、以及块状呈现：

/* 伪类 */
 {
  content: '';
  display: block;
  position: absolute;
  background-color: #fff;
}

这样他们就可以形成白色的小方块，然后通过调节具体的位置、大小、以及边框弧度加上旋转就可以实现章鱼猫的各个部位，以左边的耳朵为例：

.github > i::before {
  width: 40px;
  height: 40px;
  top: 35px;
  left: 40px;
  border-radius: 15% 70%;
}

手比较特殊，因为是异形的，实现的方式是先做一个圆形，然后在弧度最大的边添加一个比较大像素的边框，左右部分用不同像素的透明边框调节弧度：

.github > i > i::after {
  width: 28px;
  height: 28px;
  border-radius: 50%;
  top: 142px;
  left: 21px;
  border-top: 9px solid #fff;
  border-left: 1px solid transparent;
  border-right: 12px solid transparent;
  background: transparent;
  transform: rotate(39deg);
}

最后用同样的方式，画两个长方形，调整成椭圆形，然后旋转合成一颗爱心。

TCP vs UDP

2021-05-20

TCP和UDP都属于TCP/IP族内的协议，他们在传输层，负责对报文进行分组和重组，并以TCP或者UDP协议格式进行封装。

以下是两组协议的分类区分：

定义

  • TCP

Transmission Control Protocol

  • UDP

User Datagram Protocol 用户数据报协议

连接状态

  • TCP

面向连接

需要先通过三次握手建立连接再进行数据传输，传输完毕后再通过三次挥手结束连接

  • UDP

无需连接（connectionless）

数据排序

  • TCP

提供数据排序

  • UDP

无法对数据进行排序

传输保证

  • TCP

可靠传输，使用流量控制和拥塞控制

提供错误检测

  • UDP

不可靠传输，不实用流量控制和拥塞控制

只用checksum检查基本的错误

传输方式

  • TCP

面向字节流

一对一传输

  • UDP

面向报文

有单播、多播、广播的功能

首部开销

  • TCP

首部最小20字节，最大60字节

  • UDP

首部开销小，仅八个字节

传输速度

  • TCP

慢

  • UDP

快

应用场景

  • TCP

HTTPS/HTTP/SMTP/POP/FTP

  • UDP

视频会议/直播/DNS/VoIP
